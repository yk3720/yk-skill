# アクセシビリティテスト（Playwright）

## 2 層

1. **設計時** — `getByRole` / `getByLabel` でロケーターを書く（a11y ツリーと一致）
2. **任意の監査** — `@axe-core/playwright` で WCAG 違反を自動検出

## axe の位置づけ

- ロケーター選択の代替ではない
- キーボード trap・フォーカス復帰は **手動 E2E** が必要
- 動的 UI は **操作 → 安定待ち → analyze()**

詳細 → [Accessibility testing | Playwright](https://playwright.dev/docs/accessibility-testing)
