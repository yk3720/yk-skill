# アクセシビリティテスト（Playwright）

## 2 層

1. **設計時** — `getByRole` / `getByLabel` でロケーターを書く（a11y ツリーと一致）
2. **任意の監査** — `@axe-core/playwright` で WCAG 違反を自動検出

## axe の位置づけ

- ロケーター選択の代替ではない
- キーボード trap・フォーカス復帰は **手動 E2E** が必要
- 動的 UI は **操作 → 安定待ち → analyze()**
- **axe 合格 ≠ WCAG 2.2 AA 合格** — 横断 SSOT: `c:/yk-skill/rule/10_meta/A11Y_RULES.md` No **24**

## axe WCAG タグ（累積指定 · SSOT）

`wcag22aa` のみでは 2.0/2.1 由来ルールが漏れる。**6 タグすべて** を指定する。

```typescript
export const WCAG_AXE_TAGS = [
  "wcag2a",
  "wcag2aa",
  "wcag21a",
  "wcag21aa",
  "wcag22a",
  "wcag22aa",
] as const;
```

実装 SSOT（導入後）: `flowchart-studio/e2e/helpers/a11y.ts`
