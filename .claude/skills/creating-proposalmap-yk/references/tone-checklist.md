# ProposalMap — 文体チェックリスト

公開前にすべて確認する。

## 構成

- [ ] notice 帯に周知用・決裁依頼でない旨がある
- [ ] 3分要約が5行以内
- [ ] 背景→目的→作ろうとしているもの→境界→進捗→用語→お願いの順
- [ ] 「できる／今回しない」が2列表である

## プレーンディスク

- [ ] 専門用語は初出で1行定義
- [ ] 本文の説明は業務語が先、技術語は補足
- [ ] 能動態中心（「〜されます」の連鎖を避ける）
- [ ] 1文はおおよそ40字前後を目安

## です・ます（フォーマル）

- [ ] です・ますで統一（である調に混在しない）
- [ ] スラング・口語（マジで、〜っぽい）なし
- [ ] 過度な敬語（「〜させていただく」の連発）なし
- [ ] 「！」の乱用なし

## 周知向け

- [ ] お願いが「ご覧ください」「ご意見歓迎」等で、承認・予算依頼になっていない
- [ ] 読者が持ち帰るメッセージが1文で再掲されている
