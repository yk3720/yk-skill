# Web Research

公式ドキュメントを優先しながら多言語・再帰的に調査し、十分な情報が得られたら報告する。

## 調査ワークフロー

### Step 1: 初回検索（日本語・英語）

- 日本語と英語の両方で検索する
- 公式ドキュメント・仕様を最優先で取得する

### Step 2: スコアリング

検索のたびに現在の調査スコアを評価する。

**スコア基準** → [references/scoring.md](references/scoring.md)

スコアが10点に達したら検索をやめて Step 4 へ進む。

### Step 3: 再帰的深掘り

スコアが10点未満の場合、以下を繰り返す:

1. **不足している観点を特定する**（スコア基準と照合）
2. **追加言語で検索する**（中国語・韓国語・ドイツ語・フランス語など）
3. **関連キーワードで検索する**（類義語・周辺概念・エラーメッセージ）
4. スコアを再評価 → 10点に達したら終了、未達なら繰り返す

### Step 4: 報告

以下の形式で報告する:

```
## 調査結果：{トピック}

{内容の要約}

---

### 参照URL
- [{サイト名}]({URL}) — {一言メモ}
- ...
```
